# 🚀 MediTrust Quick Start Card

## Every Time You Start Working:

### 1️⃣ Double-Click This File:
```
START_PROJECT.bat
```

### 2️⃣ Wait for 2 Windows to Open:
- ✅ "MediTrust Backend" window
- ✅ "MediTrust Frontend" window

### 3️⃣ Browser Opens Automatically
- URL: http://localhost:3000
- Login and start working!

---

## ⚠️ IMPORTANT RULES:

### ✅ DO:
- Keep the 2 server windows open
- Run `START_PROJECT.bat` every time you start
- Wait 10-15 seconds for everything to load

### ❌ DON'T:
- Close the server windows
- Close VS Code during demo
- Panic if you see "Failed to fetch" (just restart)

---

## 🆘 Quick Fixes:

| Problem | Solution |
|---------|----------|
| "Failed to fetch" | Run `START_PROJECT.bat` again |
| Port in use | Script fixes it automatically |
| MongoDB not running | Run `setup-mongodb.bat` first |
| Servers won't start | Close all terminals, run script again |
