## Quick Start (3 Steps)

### Step 1: Add Your First Products
1. Login to the dashboard
2. Go to **Inventory** section
3. Click **"Add New Product"**
4. Add 5-10 products to start

### Step 2: Create Your First Order
1. Go to **Orders** section
2. Click **"Create New Order"**
3. Select products and quantities
4. Complete the order

### Step 3: View Your Dashboard
1. Go back to **Dashboard**
2. See your real data:
   - Product count
   - Today's sales
   - Sales graph (after a few orders)
