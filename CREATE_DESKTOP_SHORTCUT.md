# 🖥️ Create Desktop Shortcut for Easy Access

## Why Create a Shortcut?

Instead of navigating to your project folder every time, you can start MediTrust with **one click** from your desktop!

---

## 📝 Step-by-Step Instructions:

### Method 1: Right-Click Method (EASIEST)

1. **Open your project folder**:
   ```
   C:\Users\unish\FYP project
   ```

2. **Find the file**: `START_PROJECT.bat`

3. **Right-click** on `START_PROJECT.bat`

4. **Select**: "Send to" → "Desktop (create shortcut)"

5. **Done!** You now have a shortcut on your desktop

6. **Optional**: Rename the shortcut to "Start MediTrust"

---

### Method 2: Manual Shortcut Creation

1. **Right-click** on your desktop

2. **Select**: "New" → "Shortcut"

3. **In the location field**, paste:
   ```
   C:\Users\unish\FYP project\START_PROJECT.bat
   ```

4. **Click**: "Next"

5. **Name it**: "Start MediTrust"

6. **Click**: "Finish"

---

## 🎨 Customize Your Shortcut (Optional)

### Change the Icon:

1. **Right-click** the shortcut → "Properties"

2. **Click**: "Change Icon"

3. **Browse** and select an icon you like

4. **Click**: "OK" → "Apply"

---

## 🚀 How to Use Your Shortcut:

### Every Time You Want to Work:

1. **Double-click** the desktop shortcut

2. **Wait** 10-15 seconds

3. **Browser opens** automatically

4. **Start working!**
