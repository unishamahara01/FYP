## 🆘 Emergency Quick Fixes

### Copy-Paste These Commands:

```bash
# Start everything
.\START_PROJECT.bat

# Fix "Failed to fetch"
.\START_PROJECT.bat

# Kill stuck processes
npx kill-port 3000
npx kill-port 3001

# Setup MongoDB
.\setup-mongodb.bat

# Check if servers running
netstat -ano | findstr ":3000 :3001"

# Check if MongoDB running
tasklist | findstr mongod
```
